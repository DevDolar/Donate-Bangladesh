Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.

How do you select and element from the DOM?

There are several way to select an element from DOM. Ex.

  • Select by ID
  • Select by Class Name
  • Select by CSS selector (first match)
  • Select all matching elements with CSS selector
What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique where you use event bubbling to manage events on a parent element instead of the element where the event happens. This allows you to handle events for multiple child elements from a single place higher up in the DOM.

How do you manipulate an element's attributes and styles using the DOM?

The classList object only offers methods to change the class of an element. If you want to modify other attributes like id, href, or src, you can use the setAttribute() method instead.
